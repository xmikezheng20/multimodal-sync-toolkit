# LED Buzzer Demo

This sketch drives an LED and a buzzer together for simple audio-video demo recordings.

Behavior:

- LED and buzzer on for 1 second.
- LED and buzzer off for 3 seconds.
- Repeat every 4 seconds.

Place the LED in the camera field of view and place the buzzer near the microphone. The resulting recording can be used later as a simple visible/audible sync demonstration.
